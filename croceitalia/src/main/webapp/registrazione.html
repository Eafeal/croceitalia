Da un file Excel ad un file XML

730-PRECOMPILATO-FROM-EXCEL-TO-XML è la soluzione web, ideata da ASSOCONS SRL, che, partendo da un semplice file excel (compilato a mano o estratto automaticamente dai vostri gestionali interni), consente la generazione del file XML necessario per assolvere agli obblighi relativi al decreto legislativo 21 novembre 2014 n.175.

Scopri com'è semplice e decidi con calma

Registrati e avrai una settimana di tempo per provare gratuitamente la nostra soluzione e scoprire come è facile generare il file XML con i tuoi dati.
Al termine della prova, se sarai soddisfatto, protrai decidere se sottoscrivere la licenza d'uso del nostro servizio.
