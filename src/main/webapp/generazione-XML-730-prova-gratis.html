Da un file Excel ad un file XML

Scopri com'è semplice


730-PRECOMPILATO-FROM-EXCEL-TO-XML è la soluzione web, ideata da ASSOCONS SRL, che consente la generazione del file XML necessario per assolvere agli obblighi relativi al

decreto legislativo 21 novembre 2014 n.175

partendo da un semplice file excel compilato a mano o estratto automaticamente dai vostri gestionali interni.

Segui attentamente queste semplici istruzioni e prova a generare il tuo flusso XML:


1 - Clicca qui per scaricare sul tuo computer il modello del file excel da compilare (*)

Il file Excel proposto qui sopra fornisce il modello che consente di inserire tutti i dati previsti dalle specifiche tecniche AVCP organizzandoli in maniera semplice e lineare.

2 - Modifica il file excel inserendo i dati della tua azienda


3 - Seleziona il file excel che hai compilato e inviacelo per l'elaborazione





(*) Clicca qui per scaricare un esempio di file excel già compilato
che puoi utilizzare come ulteriore esempio per la compilazione dei tuoi dati e/o utilizzare per le successive prove di generazione.
